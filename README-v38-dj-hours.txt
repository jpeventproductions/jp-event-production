JP Event Production v38 - DJ Hours Pricing

Updates:
- Added the question "How many hours will the DJ play?" for weddings, DJ/private parties, and any request that selects DJ setup.
- Wedding minimum now includes DJ coverage up to 4 hours.
- Wedding DJ setup still does not add a separate DJ package charge.
- Additional wedding DJ playing time beyond 4 hours adds $300/hr.
- Non-wedding DJ quotes now use the DJ play-hours answer with a 3-hour minimum.
- Transparent pricing guide updated to explain the DJ play-hours and wedding included-hours rules.
