JP Event Production - Build Your Own Event v4

How to open:
1. Unzip the folder.
2. Double-click index.html.
3. Click Build Your Setup.
4. Choose one or more base layers.
5. Complete the dropdowns that appear.

What changed in this version:
- Expanded mode now starts with Build Your Own Event only.
- Customers do not choose a package first.
- Header menu is now Build, Portfolio, and Sign In.
- Copy has stronger sales pitch language and clearer tag lines.
- The parallax description panels now include service images for live sound, DJ/lighting, and event production support.
- Portfolio now includes a professional image carousel using JP Event Production setup photos.
- Carousel includes left-to-right fading transitions, next/previous controls, dots, auto-advance, and mobile swipe support.
- Final contact information is now required before a build request can be submitted.
- The first three setup panels have softer image-to-panel fades and smoother hero/gallery transitions.
- Builder options and completed fields now have subtle motion feedback when selected.
- Base setup options now use a horizontal scroll-wheel style selector.
- Pricing pop-up removed; the temporary estimate now stays in the bottom summary only.
- Temporary estimates use pricing anchors from prior quotes, including DJ services at $300/hr, equipment rental, lighting, and setup.
- Customer-facing estimate copy now says pricing may change after details are confirmed and invites customers to ask about specials.
- Selecting any "Choose one" field opens a larger bubble-style picker.
- Scroll-wheel areas use a pinwheel-style motion effect.
- Date/time, venue, coverage, and event-specific questions now use horizontal scroll-wheel style sections.
- Suggested package names and image cards now generate from the selected service combination.
- The faded setup images no longer show overlaid service words.
- Portfolio now includes a Meet the Founder button and founder bio popup using the DJing image.
- Hero copy returns to the original midsize-event positioning.
- Hero background images now use real service photos focused on speakers, stage coverage, and event rooms.
- The choice picker popup is constrained to the current browser window and scrolls internally.
- Suggested package recommendations now appear as bubbles inside the quote area.
- The base layer uses clickable equipment/service options.
- Date, time, venue size, guest count, speaker systems, power, and indoor/outdoor questions appear after a base option is selected.
- Event-specific questions appear only after event type is chosen.
- Wedding flow focuses on speaker-system count, room/venue size, coverage zones, microphones, subs, and uplighting.
- Wedding builds show a $1,000 minimum.
- Public-facing portfolio copy no longer lists the detailed drum/backline inventory.
- Backline inventory now appears inside the form when Backline is selected, including available drum kits, snares, cymbal requests, SPD-SX style support, and rider matching.
- Optional details open in a floating panel so the main flow stays clean.
- Parallax and sticky scroll sections were added for a more premium Apple-style feel.
- The hero background fades between event visuals.
- The Sign In section previews a future client dashboard for quote status, uploads, contracts, invoices, and event progress.

Production note:
The form currently saves requests to browser localStorage for demo purposes. The next step is connecting it to Supabase and email alerts.
